The function defined by 
$$F(x)=P(X \leq x) = \int_{-\infty}^x f(t)d(t),

\quad-\infty < x < \infty$$
is called the **cumulative distribution function (CDF)**
- Also referred to as the **distribution function** of a random variable $X$

$$E[X]=\int_{-\infty}^{\infty}x\dot f(x)dx$$
$$\sigma^2=Var(X)=E[(X-\mu)^2]=\int_{-\infty}^{\infty}(x-\mu)^2f(x)dx$$
